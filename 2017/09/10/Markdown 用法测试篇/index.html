Sep 10, 2017Markdown点击

序言：在Markdown中空格是一个十分重要的角色

标题

任何数量的 = 和 - 都可以有效果。

> 引用板块效果

列表

Markdown 支持有序列表和无序列表。(前面为输出样式，后面为代码)

无序列表使用星号、加号或是减号作为列表标记：

  • Red * Red
  • Green * Green
  • Blue * Blue

等同于：

  • Red + Red
  • Green + Green
  • Blue + Blue

也等同于：

  • Red - Red
  • Green - Green
  • Blue - Blue

有序列表则使用数字接着一个英文句点：

  1. Bird 1. Bird
  2. McHale 2. McHale
  3. Parish 3. Parish

如果列表项目间用空行分开，在输出 HTML 时 Markdown 就会将项目内容用

标签包起来 

*   Bird  
*   Magic

转换结果：

<ul>
<li>Bird</li>
<li>Magic</li>
</ul>
